"We spot Sintra" is a website including branding for a bouldering event as part of a UX/UI bootcamp exercise.

Project

"We spot Sintra" is a website design exercise within my UX/UI bootcamp.

The website should present an event and - a at least mid-fidelity - think throug a user flow during the event. "We spot Sintra" is based on a real festival happening in Sintra Region, Portugal.

The major aim of this website is not only ticket sales, but also the ability for volunteers to participate. In addition during the festival provides participants with a map and information of the boulders to climb.

This way the website serves as community participation possiblity pre festival and as a digital guide during and post festival..

Info

Scope

  • One high-fidelity user-flow of an event
  • Application of design system elements
  • Branding (Moodboard, Logo, ...)

Tools & Techniques

  • UX Research: Desk Research, Quantiative Research, Interviews
  • Insight Generation: Persona, Affinity Diagram, Concept Generation / Testing
  • Prototype: Adobe XD, Adobe Illustrator
